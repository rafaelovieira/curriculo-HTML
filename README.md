# Currículo Web – Rafael Vieira

Este projeto é um **currículo profissional em formato de página web**, desenvolvido para apresentar minha trajetória, competências técnicas e projetos de forma moderna e acessível online.

A aplicação foi construída utilizando **HTML, CSS e Bootstrap**, com foco em **layout responsivo, organização semântica do conteúdo e apresentação clara das informações profissionais**.

## 🌐 Demonstração

A aplicação está publicada utilizando **GitHub Pages**:

**Acessar:**
https://rafaelovieira.github.io/curriculo-HTML

## 📌 Objetivo do Projeto

Este projeto foi desenvolvido com os seguintes objetivos:

* Criar uma versão web do currículo profissional
* Praticar conceitos de **estruturação semântica em HTML**
* Aplicar **Bootstrap para criação de layouts responsivos**
* Organizar informações profissionais de forma clara e visual
* Publicar uma aplicação estática utilizando **GitHub Pages**
* Disponibilizar um portfólio simples para processos seletivos

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons
* Git
* GitHub Pages

## ✨ Funcionalidades

A página apresenta:

* Cabeçalho com informações de contato e redes profissionais
* Resumo profissional
* Competências técnicas organizadas por área
* Projetos desenvolvidos com links para repositórios
* Experiência profissional
* Formação acadêmica
* Cursos e certificações
* Idiomas

Além disso, o projeto possui:

* Layout responsivo
* Ícones de tecnologias
* Efeitos visuais em cards
* Estrutura organizada para fácil leitura
